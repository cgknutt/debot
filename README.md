# Debo

My first iOS app built with Swift and SwiftUI.

## Requirements

- Xcode 14.0 or later
- iOS 16.0 or later
- Swift 5.0 or later

## Installation

1. Clone the repository
2. Open `Debo.xcodeproj` in Xcode
3. Build and run the project
